БАЗОВЫЙ ФОРМАТ:
────────────────────────────────────────────────────────────────────────────
GET http://localhost:3000/api/vacancies?{параметры}

═══════════════════════════════════════════════════════════════════════════════
1. ПОИСК ПО НАЗВАНИЮ (title)
═══════════════════════════════════════════════════════════════════════════════

Простой поиск:
  $ curl "http://localhost:3000/api/vacancies?keywords=Уборщица"
  $ curl "http://localhost:3000/api/vacancies?keywords=Программист"

Поиск по нескольким ключевым словам:
  $ curl "http://localhost:3000/api/vacancies?keywords=Программист,Python"
  $ curl "http://localhost:3000/api/vacancies?keywords=Менеджер,Продаж"

═══════════════════════════════════════════════════════════════════════════════
2. ПОИСК ПО КАТЕГОРИИ (category)
═══════════════════════════════════════════════════════════════════════════════

Поиск по категории:
  $ curl "http://localhost:3000/api/vacancies?keywords=Уборщица&searchBy=category"
  $ curl "http://localhost:3000/api/vacancies?keywords=Программист&searchBy=category"

Поиск по категории с указанием источников:
  $ curl "http://localhost:3000/api/vacancies?keywords=Уборщица&searchBy=category&sources=rabota.md,999.md"

СВОДНАЯ ТАБЛИЦА ПАРАМЕТРОВ
═══════════════════════════════════════════════════════════════════════════════

| Параметр | Описание | Примеры значений |
|----------|----------|------------------|
| keywords | Ключевые слова для поиска | Уборщица, Программист,Python |
| searchBy | Тип поиска | title, category |
| locationType | Тип локации | moldova, abroad, aboard |
| workLocationType | Фильтр по типу локации | moldova, abroad, international |
| salaryMin | Минимальная зарплата | 20000, 50000, 100000 |
| experience | Опыт работы | без опыта, 1-3, 3-6, 6+ |
| schedule | График работы | полный день, неполный день, удаленно |
| employment | Тип занятости | полная, частичная, проект |
| skills | Навыки | Python, английский, Python,JavaScript |
| source | Один источник | rabota.md, 999.md, makler.md, hh.ru |
| sources | Несколько источников | rabota.md,999.md |
| useSemanticSearch | Семантический поиск | true, false |
| limit | Количество результатов на страницу | 10, 20, 50 |
| page | Номер страницы | 1, 2, 3 |
| userId | ID пользователя для кэширования | 123, 456 |

ВАЖНЫЕ ЗАМЕЧАНИЯ
═══════════════════════════════════════════════════════════════════════════════

1. `searchBy=title` - ищет в полях `title` ИЛИ `category` (после исправлений)
2. `searchBy=category` - ищет только по полю `category`
3. `useSemanticSearch=true` - использует словарики для маппинга и парсинга с точными названиями
4. `locationType=abroad` - включает все 4 источника (включая `hh.ru`)
5. `locationType=moldova` - только 3 источника (без `hh.ru`)
6. `workLocationType` автоматически конвертируется: `abroad` → `За границей`, `moldova` → `В Молдове`
