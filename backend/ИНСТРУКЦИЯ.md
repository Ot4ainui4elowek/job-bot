Система управления режимами позволяет легко переключаться между:
• 🔧 Режимом разработки (dev) - локальный запуск
• 🚀 Режимом продакшена (prod) - запуск через Docker

═══════════════════════════════════════════════════════════════════════════
🚀 БЫСТРЫЙ СТАРТ
═══════════════════════════════════════════════════════════════════════════

ШАГ 1: Инициализация проекта
─────────────────────────────
$ npm run init

Эта команда создаст необходимые файлы и проверит зависимости.

ШАГ 2: Проверка режима
──────────────────────
$ npm run mode

Покажет текущий режим и конфигурацию.

ШАГ 3: Настройка окружения
──────────────────────────
Откройте файл .env.local и настройте:
• DATABASE_URL - подключение к вашей PostgreSQL
• REDIS_HOST и REDIS_PORT - подключение к Redis

ШАГ 4: Запуск приложения
────────────────────────
$ docker-compose up -d postgres redis
$ npm run db:migrate
$ npm run dev:api

API будет доступен на: http://localhost:3001
